# Aneuploid_CRISPR_Screens
Data repository for scientific article about paired CRISPR screens between aneuploid and euploid cells.

"Code" Folder: this folder contains the R code files used to analyse data and produce graphs in this article. 

"Data" Folder: this folder contains count files and MaGECK CRISPR screen analysis files for the paired CRISPR screens performed for this article. This folder also contains the Mass Spec proteomics files produced for this paper. 

"Dependency" Folder: This folder contains datasets analysed and used in this article to produce figures and analysis'. Data from DepMap Cancer Cell Line Encyclopedia (CCLE), The Human Protein Atlas (HPA), The Cancer Genome Atlas (TCGA), and data files from previous publications including Magesh et al.(2025), Zerbib et al. (2024), Bokenkamp et al.(2025), Durrbaum et al. (2014), Dephoure et al. (2014), and Stingele et al. (2012). Data in this folder was not generated for this article, and most files were not generated by authors of this article. These are dependency files from other works. They are provided here for reproduciblity and academic use. 


TO RUN CODE: 
To run all the code and produce figure graphs, download the "Code" files, the "Data" files and the "Dependency" files. All code is written in R. 
ADDITIONALLY: please download "CRISPRGeneEffect.csv" CRISPR dropout data from DepMap data (Downloaded November 9, 2023), "OmicsExpressionProteinCodingGenesTPMLogp1.csv" RNA expression data from DepMap (DepMap, Broad (2024). 24Q4), and "mmc2.xlsx" DepMap proteomics data (Downloaded august 2020). These files were too big to be uploaded to Github. 
Note: "OmicsExpressionProteinCodingGenesTPMLogp1.csv" and "mmc2.xlsx" are only needed for Supplementary Figure 7, DepMep RNA and Proteomics analysis. You can run most of the code without these files. CRISPRGeneEffect.csv is used in multiple figures though. 


TO LOOK UP INFO IN ANALYZED DATA: 
I would recomend downloading the Supplementary Data files, which contain analyzed CRISPR screen results and analyzed proteomics data.
